Pandas - Missing Data

Summary
-- my_dataFrame.dropna
-- my_dataFrame.fillna

Let's deal with missing values

import numpy as np
import pandas as pd

np.nan signifies missing or null value

my_dict = {'A':[1, 2, np.nan], 'B':[5, np.nan, np.nan], 'C':[1, 2, 3]}

my_dataFrame = pd.DataFrame(my_dict)

my_dataFrame
prints out its values:

    A     B    C
0   1.0  5.0   1
1   2.0  NaN   2
2   NaN  NaN   3


Drop Na:


my_dataFrame.dropna(): Panda will drop all rows with 1+ NaN values
-- again, by default, axis =0 aka it will do rows

my_dataFrame.dropna(axis=1): Will drop all columns with 1+ NaN values

threshold
my_dataFrame.dropna(axis=1, thresh=2) # will drop all columns with at least 2 NaN values



Fill Na:

my_dataFrame.fillna(value='Fill Value')  # Replaces all NaN values with 'Fill Value'

You could even use it like this:
my_dataFrame['A'].fillna(value = my_dataFrame['A'].mean())
# This fills all the NaN values in the 'A' column with the mean for that column
