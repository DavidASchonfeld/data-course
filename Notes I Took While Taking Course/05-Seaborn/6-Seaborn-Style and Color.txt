plt.figure(figsize = (12, 3))

Seaborn is using that Matplotlib funciton (of talking about canvas size) So Seaborn works with Matplotlib

This is what shift + Tab (in Jupyter Notebook) shows us about this method:
sns.set_context(context = None, font_scale =)
Parameters:
context: dict, None or one of {paper, notebook, talk, poster}.
    dictoinary of parameters or the name of a preconfigured set
font_scale: float, optional,
    Separate scaling factor to independently scale the size of font elements
rc: dict, optional:
    Parameter mappings to override the values in the preset seaborn context dictionaries. This only updates parameters that are considered part of the context definition.

Colors
#####################################




From Previous Lectures:

sns.lmplot(x = 'total_bill' y = 'tip', data = tips, hue = 'sex')
--this automatically color-codes the data into the category designated in the hue parameter. Here, it automatically codes blue for male, green for female, draws separate linear regression lines for them too.


#Also, from previous lectures
We can pass into the "palette" parameter to choose from some preset color schemes
sns.lmplot(x = 'total_bill' y = 'tip', data = tips, hue = 'sex', palette = 'coolwarm')


Let's google matplotlib's colormap's documentation's webpage


We can see a list of all of the preset color gradients, and the string we need to pass in to get the specific gradient color that we want

For example, let's use the colormap (aka the color theme) called "seismic"

sns.lmplot(x = 'total_bill' y = 'tip', data = tips, hue = 'sex', palette = 'seismic')
